# متتبع المصاريف (Expense Tracker)

تطبيق ويب بسيط وعصري لتتبع المصاريف الشخصية، مصمم ليعمل بكفاءة على جميع الأجهزة مع تركيز خاص على تجربة الجوال.

## المميزات

- **تصميم عصري (Glassmorphism)**: واجهة مستخدم جذابة مع دعم الوضع الداكن.
- **متجاوب تماماً (Mobile First)**:
  - تحويل الجداول إلى بطاقات أنيقة على الجوال.
  - زر إضافة عائم (FAB) لسهولة الاستخدام بيد واحدة.
  - تصميم مدمج (Compact) للاستفادة القصوى من الشاشة.
- **حفظ البيانات**: يتم حفظ جميع المصاريف تلقائياً في المتصفح (LocalStorage)، فلا داعي للقلق من ضياع البيانات عند التحديث.
- **تصدير البيانات**: إمكانية نسخ سجل المصاريف كاملاً إلى الحافظة بضغطة زر لمشاركته أو حفظه في مكان آخر.
- **يدعم اللغة العربية**: واجهة عربية بالكامل (RTL) مع خطوط مريحة للقراءة (Cairo Font).

## التقنيات المستخدمة

- **HTML5**: هيكلية الصفحة.
- **CSS3**: التنسيقات والتأثيرات البصرية (بدون إطارات عمل).
- **JavaScript (Vanilla)**: المنطق والتعامل مع البيانات.

## كيفية الاستخدام

1. افتح الملف `index.html` في أي متصفح حديث.
2. اضغط على زر "إضافة مصروف" لإدخال عملية جديدة.
3. أدخل المبلغ والسبب ثم احفظ.
4. يمكنك عرض المصاريف، حذفها، أو نسخ السجل كاملاً باستخدام زر النسخ في الأعلى.

## التطوير

تم تطوير هذا المشروع ليكون بسيطاً وسريعاً، ويمكن التعديل عليه بسهولة لإضافة المزيد من الميزات مثل التصنيفات أو الرسوم البيانية.
